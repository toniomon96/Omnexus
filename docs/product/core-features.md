# Core Features

## Workout Programs

Omnexus supports both pre-built and generated programs. Users can follow a scheduled plan, inspect week/day structure, continue blocks with a progression report, or build a custom program manually. AI generation is backed by deterministic fallback logic so the training loop still works when the AI route fails.

## Exercise Library

The exercise library includes 300+ exercises grouped across multiple equipment categories. Users can browse by search, movement pattern, muscle, equipment, or difficulty, then open a detailed view with technique guidance, common mistakes, variations, cues, and related movement intelligence.

## Workout Logging

Workout logging is centered on the active workout flow, quick logging, and history review. Users can log sets with reps, load, and RPE, receive PR detection and completion feedback, then revisit sessions through history cards and charts. Guest users can use the same loop locally before upgrading to cloud sync.

## Insights

The insights system combines four-week training analysis, post-session adaptation guidance, peer benchmarking, and pre-workout briefing support. Its role is to turn raw workout history into next-action guidance rather than passive reporting.

## Learning System

The learning system packages structured course content, quizzes, daily challenges, and spaced repetition review into a retention loop that complements training. It uses static course content for reliability, while AI-generated micro-lessons and recommendations add dynamic depth.

## Ask Omnexus AI

Ask Omnexus is the conversational AI surface. It supports coaching, science, and check-in style interactions, uses citations where appropriate, and is backed by server-side prompt-safety checks. It is positioned as a coaching layer around the rest of the product, not a standalone chatbot.

## PubMed Research Feed

The research feed is the evidence-oriented discovery surface. Serverless routes proxy PubMed content, the client caches article responses, and the feed supplies source material for both direct reading and AI-grounded responses.
